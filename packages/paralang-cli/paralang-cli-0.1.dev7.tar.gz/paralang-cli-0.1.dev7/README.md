## Docs

Our documentation can be found [here](https://para.readthedocs.io/en/latest/).

## Installation

```bash
python3 -m pip install -U parac-ext-cli
```

*With specific version*:
```bash
python3 -m pip install -U parac-ext-cli==version
```
